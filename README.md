# Quantum-algorithm
#Проект: Симуляция квантового алгоритма Гровера
Библиотеки: `complex.h`, `math.h`, `stdlib.h`, `string.h`

Структура проекта:
- `grover.h/c` – реализация алгоритма Гровера (гейт Адамара)
- `main.c` – основная программа Гровера с заданными n (для гейта Адамара)

Было улучшено время для гейта Адамара с $O(N^2)$ до $O(count * n)$.
